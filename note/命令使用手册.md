#### DVC使用手册
data version control:
```
1.      git init
2.      dvc init
3.      git commit -m "initialize DVC"
以上三步为git和dvc的初始化
```

```
4.      dvc add data/data.xml
5.      git add data/.gitignore data/data.xml.dvc
6.      git commit -m "add source data to DVC"
以上三步为跟踪数据文件以及提交数据文件
```
